# HTML Website Structure

This project demonstrates a basic HTML5 webpage structure with essential elements.

## File Structure

project/
│── index.html    # Main HTML file
│── README.md     # Project documentation

## Features

- Proper HTML5 document structure with `<!DOCTYPE html>`
- Semantic HTML elements:
  - `<header>` for the page title
  - `<nav>` for navigation links
  - `<main>` for primary content
  - `<footer>` for contact information
- Well-formatted and commented code
- Responsive meta tag for mobile devices

## Requirements

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Sublime Text, Atom, etc.)

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Explore the HTML structure

## Customization

To modify this template:

1. Edit the title in the `<head>` section
2. Update navigation links in the `<nav>` section
3. Replace placeholder text in the `<main>` section
4. Change contact information in the `<footer>`

## Best Practices Followed

- Proper indentation for readability
- Semantic HTML5 elements
- Descriptive comments
- Valid HTML structure
